Description

The following tables describe the evolution of the Affymetrix data files by showing which versions of software created which versions of each data file. There are also tables to show the versions of each file type.

MAS5

File Type Version Format
EXP 1 ASCII
DAT 1 Binary (little endian)
CEL 3 ASCII
CHP 12 Binary (little endian). This is the historical format as described in the CHP file format.
CDF 1, 2, 3 ASCII
PSI No version number. ASCII
MSK No version number. ASCII
