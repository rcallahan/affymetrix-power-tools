Introduction

This vignette describes the process for using apt-copynumber-onco-ssa and apt-copynumber-onco-som-ssa to process FFPE samples run on OncoScan arrays in matched-normal or single sample mode.

Library Files

The following library files are required for analysis of OncoScan data.

  OncoScan.r1.snplist.txt               - SNP probes list
  OncoScan.FFPE.na33.r1.REF_MODEL       - Reference model file for apt-copynumber-onco-ssa
  OncoScan.Ref103.na33.r1.SOM_REF_MODEL - Reference model file for apt-copynumber-onco-som-ssa
  OncoScan.na33.r1.annot.db             - Annotation file for OncoScan arrays
  OncoScan.r1.chrXprobes                - X chromosome probes
  OncoScan.r1.chrYprobes                - Y chromosome probes
  OncoScan.Som1.0.r1.Som_thresh.txt     - Thresholds for somatic mutation calls
