Introduction

apt-copynumber-onco-som-ref implements the somatic mutation reference model pipeline for OncoScan arrays which produces a Reference Model File which can then be used by apt-copynumber-onco-som-ssa for single-sample somatic mutation analysis of OncoScan experiments. Inputs consist of CEL files for a number of OncoScan experiments (each experiment consists of two CEL files, one for the AT and one for the GC channels), and additional library and annotation files.

apt-copynumber-onco-som-ref requires a minimum of six experiments, including at least one male and one female sample, to run; however a minimum of 22 male and 22 female samples is recommended.

Quick Start

Input/Output Files

The following input files are required by apt-copynumber-onco-som-ref:

Name File extension/type Description
spf-file spf5 (HDF5) Describes probes and probesets on array.
som-spf-file spf5 (HDF5) Describes somatic mutation probesets on array.
annotation-file annot.db (SQLite) Provides annotation data for probesets.
x-probes-file chrXprobes (text) Probes for X chromosome (used in raw gender calling)
y-probes-file chrYprobes (text) Probes for Y chromosome (used in raw gender calling)
cel-pairs-file txt CEL file pairs (AT and GC channel per sample) for reference generation

The following output files are generated by apt-copynumber-som-ref:

Name File extension/type Description
reference-file SOM_REF_MODEL (HDF5) The reference model file for the Copynumber probesets

CEL files lists:

The CEL file pairs are specified in a tab delimited text file with two columns representing the AT and GC-channel CEL files respectively. The first row of the cel-pairs-file should contain the column names atchannelcel and gcchannelcel.

See OncoScan reference model generation for examples.
